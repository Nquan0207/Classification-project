📊 Classification Metrics Formulas

Accuracy
Accuracy = TP + TN TP + TN + FP + FN = Correct Predictions Total Predictions

Overall correctness across all classes.

Precision
Precision = TP TP + FP = True Positives True Positives + False Positives

Proportion of predicted positives that are correct.

Recall
Recall = TP TP + FN = True Positives True Positives + False Negatives

Proportion of actual positives identified correctly.

F1-Score
F1 = 2 x Precision x Recall Precision + Recall = 2TP 2TP + FP + FN

Harmonic mean between precision and recall.

Top-5 Accuracy
Top-5 Accuracy = Correct Top-5 Predictions Total Predictions

A prediction is counted correct if true label appears in top 5 outputs.

📈 Averaging Methods

Macro Average
Macro-Metric = 1 N x Σ(Metric_i)

Treats every class equally, regardless of class size.

Weighted Average
Weighted-Metric = Σ(Metric_i x Support_i) Σ(Support_i)

Weights each class metric by number of true instances.

⚡ Resource Metrics

FLOPs
FLOPs_total = Σ(FLOPs_layer), with FLOPs_layer ≈ 2 x MACs

Approximate compute cost for one forward pass.
